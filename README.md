## 📌 Project Overview
This project focuses on **parsing and formatting nested JSON data** using Python.  
It demonstrates how to read a JSON file, extract required information, and solve multiple query-based tasks using loops, lists, and dictionaries.

The notebook contains **20 practice questions** based on real-world nested JSON structures:
- Company details
- Departments
- Employees
- Skills
- Salaries
- Locations

---

## 🎯 Objectives
- Load and read nested JSON data using Python
- Extract values from multi-level JSON structure
- Perform analysis and transformation
- Create output using:
  - Lists
  - Dictionaries
  - Looping logic
  - String operations
